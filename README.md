# 📘 High-Intensity SQL Boot Camp – README

## Overview

The **High-Intensity SQL Boot Camp** is a one-week, academically grounded and industry-focused training program designed to build **robust SQL skills** for:

- 💼 Job readiness (junior developer, data analyst, backend roles)
- 🌐 Web development (database design, backend integration, CRUD operations)
- 🎓 Strong academic foundation in relational database theory

This boot camp balances **theory + applied practice**, ensuring students understand _why_ SQL works — not just how to write queries.

---

# 🎯 Program Objectives

By the end of this boot camp, participants will be able to:

- Design relational database schemas using normalization principles
- Write complex SQL queries confidently
- Use JOINs, subqueries, and aggregations effectively
- Apply indexing and constraints appropriately
- Solve real-world business and web application problems
- Prepare for SQL technical interviews
- Integrate SQL thinking into backend and web development workflows

---

# 🧠 Academic Foundation

This program is grounded in core database theory, including:

- Relational Model concepts
- Primary & Foreign Keys
- Entity relationships (1–1, 1–many, many–many)
- Normalization (1NF, 2NF, 3NF)
- ACID properties
- Constraints and data integrity
- Indexing fundamentals
- Query optimization basics

Participants will gain conceptual clarity alongside applied skills.

---

# 💻 Job & Web Development Readiness

The boot camp prepares learners to:

- Query and analyze production databases
- Design backend-ready schemas
- Write optimized JOIN queries
- Implement reporting queries (revenue, rankings, metrics)
- Think in terms of scalable data systems
- Translate business questions into SQL logic

Real-world use cases include:

- E-commerce systems
- Blog platforms
- School management systems
- User authentication databases
- Analytics dashboards

# 📅 Program Structure (1 Week – High Intensity)

### [Day 1 – Database Fundamentals](./DayOne.md)

- Relational databases & DBMS
- Schema design basics
- Data types
- SELECT fundamentals

### [Day 2 – Filtering & Logical Operators](./DayTwo.md)

- WHERE, AND, OR
- IN, BETWEEN, LIKE
- NULL handling

### [Day 3 – Aggregation & Reporting](./DayThree.md)

- COUNT, SUM, AVG
- GROUP BY & HAVING
- Business metric queries

### [Day 4 – JOIN Mastery](./DayFour.md)

- INNER, LEFT, RIGHT JOIN
- Relationship modeling
- Multi-table queries

### [Day 5 – Subqueries & Nested Logic](./DayFive.md)

- Subqueries in WHERE and FROM
- Ranking & interview-style problems

### [Day 6 – Database Design & Indexing](./DaySix.md)

- Normalization principles
- Constraints
- Index fundamentals
- Performance awareness

### [Day 7 – Capstone Project](./DaySeven.md)

- Design a full relational schema
- Write 20–30 real-world queries
- Simulate backend/web use cases
